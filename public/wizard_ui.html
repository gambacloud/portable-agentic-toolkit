AI Model

To run agents, you need an LLM. We support local models via Ollama or fast cloud models via Groq.

Option A: Local Model (Ollama)

If you have Ollama installed, open your terminal and run:

ollama pull llama3.2

This takes a minute or two depending on your internet connection.

Option B: Cloud Model (Groq)

Edit your .env file in the project directory and add your API key:

GROQ_API_KEY=gsk_your_api_key_here

Agent Identity

Create your first Expert Profile. This gives your agent a persona, a specific role, and background knowledge to act upon.

Connectors (MCPs)

Connectors (Model Context Protocol servers) give your agent real-world capabilities. Install a couple of basic ones to get started.

🌐

Brave Search

Allows the agent to search the internet for real-time information.

📁

Filesystem

Allows the agent to read and write files on your local machine.
